# WiFi Person Detector (Ubiquiti Multi-Floor, Multi-AP)

This project detects persons using WiFi signal strength from Ubiquiti access points, supporting multi-floor and multi-AP configurations.

## Features

- Monitors signal strength from multiple Ubiquiti APs
- Detects presence and movement of persons
- Supports multi-floor setups
- Configurable thresholds and filtering

## Setup

1. Install required dependencies
2. Configure your Ubiquiti APs
3. Run the detector

## Usage

```bash
python3 detector.py
```
